# movies-project
## Install dependencies

```bash
$ cd movies-app
$ npm install
```

## Run MongoDB server

## Start api server

```bash
$ cd movies-app
$ babel-node server/app.js
```
api run on http://localhost:8080

## Start development server

```bash
$ cd movies-app
$ npm run webpack-devserver
```
app run on http://localhost:8090
